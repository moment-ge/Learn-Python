## list

list 是一种有序、可变（mutable）的集合类型，可以存储不同类型的元素，允许随时添加和删除元素。

> 类比为 JavaScript 中的 数组，两者在语法和功能上有很多相似之处。

比如，列出班里所有同学的名字，就可以用一个 list 表示：

```python
classmates = ['张三', '李四', '王五']
print(classmates)
```

`len()`函数可以获得list元素的个数

```python
classmatesLength = len(classmates)
print(classmatesLength)
```

> 索引来访问list中每一个位置的元素，记得索引是从 `0`开始的

### 访问 list 元素

通过索引访问每个同学的名字：

```python
first_classmate = classmates[0]  # 获取第一个同学
last_classmate = classmates[-1]  # 获取最后一个同学
print("第一个同学:", first_classmate)
print("最后一个同学:", last_classmate)
```

### 修改 list 中的元素

`list` 是 **可变的**（mutable），意味着你可以随时 **修改** 它的内容：

```python
classmates[1] = '赵六'
print("修改后的班级同学名单:", classmates)
```

### 添加元素到 list

你可以使用 `append()` 方法将元素添加到 `list` 的末尾，或者使用 `insert()` 在指定位置插入元素：

```python
classmates.append('孙七')
print("添加新同学后的名单:", classmates)

classmates.insert(1, '周八')  # 在索引位置 1 插入新同学
print("插入新同学后的名单:", classmates)
```

### 删除元素

你可以使用 `remove()` 或 `pop()` 方法删除元素：

```python
classmates.remove('王五')  # 删除指定的同学
print("删除 '王五' 后的名单:", classmates)

last_removed_classmate = classmates.pop()  # 删除并返回最后一个元素
print("删除的最后一个同学:", last_removed_classmate)
print("删除后的名单:", classmates)
```

### `list` 的其他常用操作

```python
# 检查一个同学是否在名单里
print("李四在名单里吗?", '李四' in classmates)

# 获取同学名单的反转版本
reversed_classmates = classmates[::-1]
print("反转后的名单:", reversed_classmates)

# 清空所有元素
classmates.clear()
print("清空后的名单:", classmates)
```

## tuple

tuple 是一种有序、不可变（immutable）的集合类型。与 `list` 不同，`tuple` 创建后不能修改、添加或删除其中的元素。

```python
my_tuple = (1, 2, 3)
# 你不能像下面这样修改 tuple
# my_tuple[0] = 10  # 会报错: TypeError: 'tuple' object does not support item assignment
```

### **tuple 的不可变性**

一旦初始化，**tuple** 中的元素就不能修改，也没有像 **list** 的 **append()** 和 **insert()** 方法。你不能直接修改元素：

```python
classmates[1] = '赵六'  # 会报错: TypeError: 'tuple' object does not support item assignment
```

### **tuple 的意义**

不可变的 tuple 有助于提高代码的安全性。如果你能用 tuple 来代替 list，它通常会更加安全，因为 tuple 是不可变的，避免了意外修改的数据错误。

### **tuple 的陷阱**

当你定义一个包含一个元素的 **tuple** 时，必须加上一个逗号 **,**，否则 Python 会认为你是定义一个普通的括号表达式。

```python
t = (1,)  # 正确
print(t)  # 输出: (1,)

t = (1)  # 错误，结果是一个整数 1，而不是一个 tuple
print(t)  # 输出: 1
```

### **“可变的” tuple**

虽然 **tuple** 本身是不可变的，但如果 **tuple** 的元素是 **可变对象**（如 **list**），那么 **可变对象的内容是可以修改的**：

```python
t = ('a', 'b', ['A', 'B'])
t[2][0] = 'X'  # 修改 list 中的元素
t[2][1] = 'Y'
print(t)  # 输出: ('a', 'b', ['X', 'Y'])
```
