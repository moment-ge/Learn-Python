# 模式匹配

> **当使用 **`if ... elif ... elif ... else ...` 判断时，代码会很长，可读性较差。

想想 javascript 中的 switch，有点类似

## match 语句

### 传统 if 语句写法

```python
score = 'B'
if score == 'A':
    print('score is A.')
elif score == 'B':
    print('score is B.')
elif score == 'C':
    print('score is C.')
else:
    print('invalid score.')
```

### 使用 match 语句改写

```python
score = 'B'

match score:
    case 'A':
        print('score is A.')
    case 'B':
        print('score is B.')
    case 'C':
        print('score is C.')
    case _:  # _ 表示匹配到其他任何情况
        print('score is ???.')
```

> `case`\_表示“任意值”

### 复杂匹配

match 语句可以：

- 匹配多个值
- 匹配一定范围
- 把匹配后的值绑定到变量

```python
age = 15

match age:
    case x if x < 10:
        print(f'< 10 years old: {x}')
    case 10:
        print('10 years old.')
    case 11 | 12 | 13 | 14 | 15 | 16 | 17 | 18:
        print('11~18 years old.')
    case 19:
        print('19 years old.')
    case _:
        print('not sure.')
```

> `|`表示“满足其中任意一个条件”
