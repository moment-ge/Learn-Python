# 条件判断

> 代码案例参考:[条件判断.py](./Code/条件判断.py)

> 计算机能做自动化的任务，因为它可以自己做条件判断。

## `if` 语句

`if` 语句用于判断一个条件是否为 `True`，如果为真，就执行对应的代码块。

```python
age = 20
if age >= 18:
    print("你是成年人")
```

## `elif` 语句

**elif** 是 “else if” 的缩写，表示在 **if** 条件不满足时，继续检查其他条件。一个 **if** 可以有多个 **elif** 语句，表示多个不同的条件。

```python
age = 15
if age >= 18:
    print("你是成年人")
elif age >= 13:
    print("你是青少年")
```

## `else` 语句

**else** 语句用于在所有 **if** 和 **elif** 条件都不满足时执行默认的代码块,需要注意不要少写了冒号 `:`。

```python
age = 10
if age >= 18:
    print("你是成年人")
elif age >= 13:
    print("你是青少年")
else:
    print("你是儿童")
```

## `input` 函数

**input()** 是一个 **内置函数**，用于从用户那里获取输入的值。这个函数会暂停程序的执行，等待用户输入一些内容，并返回用户输入的字符串。

```python
input(prompt)
```

* **prompt**：这是一个可选的字符串，作为提示信息显示给用户。通常用来告诉用户应该输入什么类型的数据。
* **返回值**：**input()** 会返回用户输入的内容，始终是字符串类型。

```python
birth = int(input('birth: '))
if birth < 2000:
    print('00前')
else:
    print('00后')

birth = int(input('birth: '))
if birth < 2000:
    print('00前')
else:
    print('00后')

```
