# 使用文本编辑器

在 Python 的交互式命令行中编写程序有一个显著的优点：可以立即看到结果。然而，缺点是无法保存这些代码，下一次运行时必须重新输入。在实际开发中，我们更倾向于将代码保存到文件中，这样可以方便地重复运行和修改。

因此，使用文本编辑器是编写和保存代码的最佳选择。

## VSCode

### 为什么选择 VSCode？

**VSCode** 是由微软推出的轻量级、跨平台的编辑器，它并不是一个庞大的 IDE（例如 Visual Studio），而是一个精简版的编辑器，非常适合前端开发人员。

### VSCode 的优势：

1. **跨平台支持**：适用于 Windows、macOS 和 Linux。
2. **前端开发者熟悉**：由于 VSCode 支持大量的前端开发插件，因此前端开发者对于 VSCode 会感到非常熟悉。
3. **丰富的插件支持**：VSCode 提供了强大的插件生态，支持 Python、JavaScript、HTML、CSS 等多种编程语言的开发，极大地提高了编程效率。
4. **内置调试器**：你可以直接在 VSCode 中调试 Python 或其他语言的代码，而不需要切换到其他工具。
5. **简洁和轻量**：即使是轻量级的编辑器，VSCode 也提供了诸多强大的功能，比如代码自动补全、版本控制集成、集成终端等，满足大多数开发需求。

## 推荐插件：Code Runner

为了在 VSCode 中方便快捷地运行代码，推荐安装插件 **Code Runner**。该插件支持多种编程语言，可以让你直接在 VSCode 中执行代码并查看输出，非常适合快速测试和运行代码。
